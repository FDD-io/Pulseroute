## 📋 Overview

**Pulseroute** is an AI-powered triage agent for healthcare. It analyzes patient symptoms and context, then automatically routes them to the most appropriate care pathway — emergency, specialist, or general practice — in under 90 seconds.

## ✨ Features

- 🧠 AI-driven symptom analysis
- ⚡ Sub-90-second patient routing decision
- 🏥 Multi-care-pathway support (emergency / specialist / GP)
- 📊 Structured patient intake data processing
- 🔌 REST API for integration with existing healthcare systems

## 📁 Project Structure

```
├── data/           # Sample patient datasets
└── docs/           # Architecture & documentation
```

## 🚀 Getting Started

```bash
git clone https://github.com/FDD-io/Pulseroute.git
cd Pulseroute

# Backend
cd backend
pip install -r requirements.txt
python main.py

# Frontend
cd frontend
npm install && npm start
```
